#  Analyse des ventes

## Objectif

Ce projet a pour objectif d’analyser des données de ventes afin de comprendre les performances des produits et des catégories, ainsi que d’identifier les éléments les plus rentables.

---

## Données

Le dataset utilisé (`ventes.csv`) contient les informations suivantes :

* Date de vente
* Produit
* Catégorie
* Prix
* Quantité

Une colonne supplémentaire **Chiffre d’Affaires (CA)** a été calculée.

---

##  Analyse

Les analyses réalisées dans ce projet sont :

* Calcul du chiffre d’affaires total
* Identification du produit le plus vendu
* Analyse des quantités vendues par produit
* Analyse du chiffre d’affaires par catégorie
* Analyse du chiffre d’affaires par produit

---

##  Visualisations

Plusieurs graphiques ont été réalisés :

* Barplot du chiffre d’affaires par catégorie
* Barplot du chiffre d’affaires par produit
* Barplot de la quantité vendue par produit

---

## Résultats 

* Les produits électroniques génèrent une grande partie du chiffre d’affaires
* Certains produits ont un chiffre d’affaires élevé malgré une faible quantité (produits chers)
* Les ventes varient fortement selon les produits
